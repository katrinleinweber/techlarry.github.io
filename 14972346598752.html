Implementing a stack

The Stack ADT(Abstract Data Types) uses a python list. The interface for
a stack consists of these operations:

  • __init__: initialize a new empty stack
  • push : add an new item to the stack
  • pop : remove and pop an item from stack
  • isEmpty : check whether the stack is empty
  • printstack : print the stack

Here is an implementing stack with python lists.

class Stack:
    def __init__(self):
       self.items = []

    def push(self, item):
        self.items.append(item)

    def pop(self):
        return self.items.pop()

    def isEmpty(self):
        return (self.items == [])

    def printstack(self):
        print self.items

Using a stack to evaluate postfix

Here, we use a stack ADT to evaluate postfix. Postfix make operator
follows operands, such as 6 7 + . Using a stack in that way:

  1. split the expressions to numbers and operators
  2. read a number or operator each once
  3. push a number on a stack after reading a number
  4. pop two numbers off stack and calculate them after reading a operator

In order to search a number, we use regular expression(re). Python
offers re module to deal with regular expression.

import re

re.search('\d+', expression)

The code will search numbers in the expression, \d denotes a number,
+ denotes more than one. Hence, \d+ denotes more than one number.

Function postfix calculate the result of expression:

#!/usr/bin/python
#Filename: postfix.py
"""
an exercise, apply postfix algorithm to the expression 1 2 + 3 *
"""
import re   
import string

class Stack:
    def __init__(self):
        self.items = []

    def pop(self):
        self.items.pop()

    def push(self, item):
        self.items.append(item)


def postfix(expr):
    if expr == "":
        print "The expression format wrong!"

    expr_split = expr.split(' ')
    expr_stack = Stack()

    for char in expr_split:
        if char == '+':
            expr_stack.push(expr_stack.pop() + expr_stack.pop())

        elif char == "-":
            expr_stack.push(expr_stack.pop() - expr_stack.pop())

        elif char == "*":
            expr_stack.push(expr_stack.pop() * expr_stack.pop())

        elif char == "/":
            expr_stack.push(int(expr_stack.pop() / expr_stack.pop()))

        elif re.search('\d+', char)!= None:
            expr_stack.push(string.atoi(char))

        else:
            print "Expression format is wrong"

    return expr_stack.items

if __name__ == "__main__":
    expression1 = "1 2 + 3 *"
    print expression1, '=', postfix(expression1)

    expression2 = "2 3 * 1 +"
    print expression2, '=', postfix(expression2)
