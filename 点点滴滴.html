title: 资料收集症–拖延症
author: imlarry
post_id: 19
date: 2014-03-03

tags: learning

资料收集症是一类拖延症。所谓拖延症，就是你明知道自己该做事了，但就是不想去做正事。资料收集症患者在浪费时光，乱点网页的过程时都会产生这样的心理：我其实根本看不完；但是患者往往还坚持趴在电脑前，疯狂的下载。

根据Klaus Wertenbroch和Dan Ariely的实验结果以及推论，人由于自然选择的结果，往往会优先考虑当前的结果，而忽视长远的利益。举例来说，你在三个月后要考托福，在学习现在的课程与托福相比较的时候，我往往优先选择上现在的课程。因为托福还有几个月呢，这么重要的事情，在今天看来都是不重要的，于是我们可以肆无忌惮的把未来的决定性的重大事件抛之脑后，先享受现在的快乐。 我们人类往往满足于当下的快乐。因此当在暂时性的快乐和未来的快乐中选择的时候，我们的选择结果往往是：去逛街吧，先看电影吧，未来总是有时间的。我们可怜的人类啊，结果呢？你懂得。

我们必须先痛苦后快乐，这是唯一的解决方案。Why? 先痛苦后快乐的这种解决方案直接破解了满足于当下快乐的人类驱动性。在时间管理中，我们也经常提到，先“List what you think important and then give yourself rewards”,其实以另一种角度看来，也是先痛苦后快乐的这个道理。还有“Each frog a moring”,也说的是这么回事。

因此，当遇到想下载东西的时候，或者想MARK得时候，首先考虑我能不能看完，有没有必要看完，我看不看不懂？

Introduction

Researchers have found that we have two fundamentaly different modes of
thinking. One is focused mode and another is diffuse mode. When you are
focused on something and trying to learn or to understand, you are in
focused mode. When you are relaxed to think something (eg. listening to
music ), you are in diffuse mode.

Analog

If you use Diffuse mode, you can learn something new. When you learn
concepts you haven't thought of before, you don't know where the pattern
is. To get to this new thought pattern, you need a different way of
thinking. That's the diffuse mode. Diffuse mode and Focused mode can be
represented by pinball game which is showed below. The blue bumpers are
closed to each other. Red pattern at the left represents a familiar
thought pattern. Red pattern at the right representing diffuse mode goes
long way, explores more bumpers and links many bumpers together.

Which mode

Learning something difficult needs a lot of time. The brain needs to
alternate its ways of learning between focused and diffused mode as it
grapples with and assimilates the new material.

Ctags使用

title: 每天学习8小时的目标不高
author: imlarry
date: 2017-01-08
tags: Studying

categories: 时间管理

首先什么是每天8小时。我指的每天8小时是存粹的学习时间达到8小时，也就是说你在学习中间的休息时间全部不算。比如说，你上课上了40min，下课了10分钟，那么只能算40min。那么就是说如果要达到学习时间8小时，其实要学习10-11小时。

为什么每天8小时。这是我自己统计出来的，每天6-7小时很轻松，9-10小时比较累。比如今天我睡到了10点，下午还去吃了个大餐。到晚上的时候，刚好有8个小时。所以8小时是一个认真学习的时间上的底线。如果每天都抽不出10小时来学习，如果每天的存粹学习时间达不到8小时，那么这一天可以认为没有认真学习。但当然不是说每天，365天都得8小时，该休息还是要休息。

什么时候会达不到8小时。一般来说，一天除了吃饭、睡觉都在学习搞科研，学习时间可以达到13-14小时，存粹的学习时间可以达到10小时。但达到10小时是非常累的，最好出去运动运动，看个美剧放松放松，这样存粹学习时间在8-8.5小时最佳。

title: 集中学习 or 分散学习
author: imlarry
date: 2016-12-04

tags: Time Manage

集中学习即在一段时间内大量学习（每天4小时以上），分散学习即每天抽出一定时间学习（每天1小时）。那么这两种学习方法哪一种更好？

首先，对于记忆，分散学习效果要好于集中学习。对于难度大的事情，集中学习更好。背单词，这样的学习肯定是分散学习效果最佳。一个单词在半年之后才能被牢记。短短几十天记住的单词在经历较长时间尺度之后，往往会忘却。对于难度大的事情，比如构思并写一个程序，最好一气呵成。GRE考试、托福考试、四六级考试、计算机考试，最好一气呵成，这些考试难度大，拖的时间越长，越痛苦。

其次，分散学习，有助于降低痛苦感，在做的事情特别艰巨的时候。采用集中学习，如在一个月内背3000个GRE单词，一个月内写好一篇文章，很可能会放弃。例如，研究者发现那些每天写一段论文的人更有可能获得终身教授的职位。而以我为例，我先前特别怕背单词，四六级都没背过，但是后来规定每天背30个单词后，我背了近2万个单词！这些都很好的说明了分散学习的优点。

但是，对于一项特定的，不需要特别长的时间的技能或能力，集中学习显然更好。比如集中训练某一题型，集中调研某一个研究方向，等等。这些能力或者知识一次性就能解决，没有必要每天抽出一点时间来搞定它们。

title: 新的时间管理框架
author: imlarry
date: 2016-08-10

tags: Time Manage

缘由

前两个计划都失败了。其实断断续续看过很多书，尝试过不同的计划，但可以看到的是计划在越来越接近于实际。况且即使在计划失败的情况下，有计划也远远比没有计划好的多。毕竟计划总赶不上变化，但是没有计划肯定是不行的。

到现在，我的时间管理其实已经进步了好多：应用了GTD、番茄时间、日程、日记、EXCEL统计五个方式来监测、管理。除非不想学习，一直看闲书、旅游，当自己静下心来的时候应用这些方法确实能够显著提高效率。

但是现在一个很明显的出错的地方：无法长时间稳定持续，换句话来说，我的计划都是“不可持续发展的”。在深层次上，我缺乏有规律的休息，调整速度慢等等重大缺点。

那么现在提出一个新的时间管理框架，所有计划都必须遵循这个框架。如果说计划是法律的话，这个可以认为是宪法。还有框架要有详细的监控实施步骤。其实计划要实行的话，还需要强大的意志力，兴趣。并且我这个时间管理框架就其实和原来的基本吻合，这里步骤里的事情都干过，但是从来没有系统的写下来。

新框架

  1. 有效学习时间8*5+7
  2. 闲书每天小于0.5小时
  3. 健身每星期至少3次
  4. 星期日早上必须睡懒觉
  5. 星期日下午必须放松
  6. 星期日晚上做每周总结、计划
  7. 每周计划条目小于4个，要写出具体小时数目
  8. 每周总计包括时间、效率。

步骤

  1. 每天早上根据每周计划，写GTD
  2. 每天晚上必须同步日程、写日记
  3. 高效完成
  4. 星期一到星期五必须做正事
  5. 星期日必须休息
  6. 一周结束必须写总结、新计划，统计时间
  7. 优先完成计划中的事情。

title: 开始和坚持
author: imlarry
date: 2016-08-10

tags: Time Manage

有些事情，最难的就是开始。特别是日常中已经做过的事情。为什么这么说呢？比如说洗衣服。我总是要堆一堆衣服然后再开始洗，但是一旦开始洗衣服，就会发现，其实很简单。比如说常规学习，有时候不想学习了，但是一旦开始，就会发现多么美妙。

为什么想写这个，因为我觉得这很重要。对于两种不同的情况，要采取不同的措施。

title: 240计划 －－ 完败
author: imlarry
date: 2016-07-04

tags: Time Manage

感觉过去这一年，研究生二年级，最大的毛病就是反反复复。什么东西都坚持不下来。比如说每次减肥总是不能超过1个月，然后就想出去吃自助。学习上每过上半个月、20天，就想休息几天，然后又得花上几天渐渐恢复起来。反反复复带来的痛苦有很多，我在先前《反复》里就详细写了。但是总是避免不了。不过好歹是认识到了错误。下一步就是改进了。

要说进步，还是有的。至少通过各种时间管理方法，感觉效率在慢慢提升。但是总体情况，与我的理想状态相比，还差很远。所以就有了一些想法。其中一个想法一直在我脑海中反复出现：我要坚持1个月，每天最低限度的有效学习时间达到8小时以上。简单的可以说成“240计划”。

看起来每天8小时真的不多。我很多个星期都达到或超过这个数目。但是加上“每天至少8小时，以及整整一个月”这两个限制性条件，我还没有达到过。应该说距离这个目标还有很大距离：总有偷懒的时候，总想放弃的时候。但是人的潜力是无限的，所以我要争取完成这个目标。

这个计划的开始时间是2016年7月1日，计划完成时间是2016年8月1日。

完成结果 －－ 完败

打完鸡血，每天有效学习时间终于达到了10小时以上，也就是说以百分之70～80的学习效率计算的话，每天要学习12～13小时。然后终于扛不住了。坚持了11天。彻彻底底失败。完败。一败涂地。

虽然失败了，但是也很有收获。

首先，分析一下失败的原因。那就是这个计划太变态啦！至少以我目前的意志力，时间管理能力，还远远不能实现这个目标。

完成这么变态的计划，为什么很难，我觉得就两个原因：第一个就是时间必须非常规律。这一点，我做的不好，导致时间浪费很多。想学习了，就使劲学习，然后累了，第二天可能达不到8小时了，再坚持，反复，恶性循环。所以规律的作息时间是提高效率的最好途径。没有之一。第二个就是要协调好时间。该健身，该跑步，该放松还是要的。每天1小时锻炼身体是基础。

经过这几天算明白了：我离我的目标还非常遥远。

理论上说，这个目标是完全实现的了的。现实就是，我现在还不能做到。

下一步

这几天休整了一下。先做一个轻松的目标计划，4*(9*6)计划：每周强制学习将为6天，但每天的学习时间必须在9小时以上，一共坚持4个星期。从今天开始执行。

title: THE POWER OF LESS
author: imlarry
date: 2016-06-15

tags: Life

半个月前看了一本书叫做《POWER OF LESS》，
说实话非常震撼。这本书其实要说的很简单：用有限的时间最最重要的事情。用俗话说，“少即是多”嘛。由此联想到先前看过的一本书《断舍离》，讲整理杂物的一本书籍，其实质就是通过不断的丢东西，留下最重要的物品，从而达到梳理人生、净化心灵等作用。

要说为什么触动到我？是因为我做事情一项贪多。我先前在博客《少就是多的原则》里写到一点，但是对于“少”的理解还不深刻。我先前认为少是：要滚动式的学习，不要临时抱佛脚。这确实是一种“少”，但是层次太低。看完这本书以后，我发现我很多问题都来源于“多”。是的，我追求太多了，而且没有从本质出来，也就是没有抓住最重要的事情。

首先，作者认为“少”是建立在重要的基础上的。确实是这样：如果做的都是不重要的事情，做的再多也没有意义。但是我很多时候，太追求多。然后我就想为什么会追求多呢？细细想来，竟然是我不知道什么是最重要的？举个身边的例子：有个研究生要毕业了，同时参加了国企面试、私企面试、考公务员、留所。这是身边确实发生的事情，但很显然他没有意识到自己将来到底要干什么！但是我自己看到了别人的例子，自己先前也没有意识到自己的问题！那么可以说，如果自己的追求很多的话，往往就是自己不知道什么是最重要的！！！现在拿自己的例子举例：研一的时候学了很多课，这也想选，那也想选，最后都学的不怎么样，其实少选几门就可以了。我那时候对很多课程都不了解，导致了浪费了很多时间。还有一件事，我在减肥的时候，花了很多时间运动，其实减肥超级简单嘛：少吃点。在我能控制我的饮食之后，体重下降的速度明显加快了。看来我先前都不了解减肥的核心啊。

其次，作者提出了6个步骤。作者所提出的这6个步骤，在其他书上似乎都出现过。但从来没有把它们归纳到一起，形成一个系统。具体步骤是：设限，选择最重要，简化，聚焦，形成习惯，从小处开始。感觉非常有道理啊！我打算把它用到学习上，先试一试，过几个月再来写试验结果。

title: 下半年健身目标
author: imlarry
date: 2016-06-15

slug: next_half_year_fitness_plan

回顾

首先，回顾一下上半年的减肥史。在新年的时候，体重在130左右，体脂率20%左右。过年前减轻到125左右。年后大概胖了2斤。然后继续减到125斤。随后4月份回家一次，又反弹2斤。所以基本上前4个月都没减肥。随后又反弹一次，有2～3斤。到5～6月份，才慢慢开始减肥下来，这几天一直稳定在122斤，体脂率17.5%。

对于这个结果，有点小失望。因为中间反弹了好几次，原因是坚持不住了，暴饮暴食（吃自助餐，吃麦当劳、肯德基等）。如果没有反弹的话，体重估计都在116附近了。

去年在《新年健身目标》里设的目标是：120~123斤，体脂率14~16%。目前看来体重达标了，体脂率没有达标。为什么呢？原因主要是反复体重反弹造成了肌肉大量流失：每一次增的都是肥肉，但是大量减肥肉的同时，不可避免减掉肌肉。虽然从最终效果看只减肥了8斤，但是实际上减了有14斤左右。

新目标

新的目标对于体重没有要求，因为现在开始慢慢进入以增肌为重心的阶段。但是对于体脂率，希望达到15%以下。饮食以高蛋白低碳水化合物为主。

title: 反复
author: imlarry
date: 2016-05-29

tags: Life

养成一个习惯是很不容易的事情。过了20天，过了半年以后仍旧会丢掉这个习惯。再次拾起，发现这些习惯真好，于是又慢慢养成这些习惯。但是会不会再次中断？

以我的经历来说很多习惯都会中断，很少有坚持一年以上的；但是大部分都有几个月，甚至半年。比如说我在一年半前有写周记，做周计划的习惯。现在想来这是非常好的一个习惯：可以总结回顾这一周的不足之处，进步的地方；也可以对下一周进行一个初步的规划，让自己总体有序。但是这个习惯在几个月的尝试之后最终放弃。至于为什么要放弃这么好的习惯，我现在也回想不起来，如果让我猜的话，那就是“一次忘了，次次忘了”。还有一个例子，比如说健身，研一坚持了半年，然后放弃半年，重新拾起，一直到坚持到现在。所以坚持一个习惯真的很难。

于是这些习惯慢慢就反复起来。很少有那种理想的习惯：就是每天（比如说早起，背单词等）都做到了，每个星期（周记，周计划，每周写一篇英文文章等）都做到了的习惯。是不是每天都坚持很难做到？但是在身边或者网络上总有这样的人：它们做到了，365天，700天每天早上起来背单词，写作文，晨练。这让我非常震惊。根本就不可能啊！大年三十，春节都在背单词？好像太过分了。但的却他们坚持下来了。看来，坚持一个习惯的却很难，但是确是完全可能的。

坚持一个习惯的好处太多。我觉得有两个非常大的好处。第一个就是不间断的重复做一件事，对于技巧或能力的提升非常大，因为它可以跨过“平台期”。而间断了的习惯，往往越不过“平台期”，要经历很长时间以后才会有质的提升。第二个就是，间断的习惯，再次重拾起来的时候要花费巨大的代价。例如过年了，你背单词中断一个15天：虽然只是15天，但是由于记忆曲线的关系，你一个月内的单词全百背了；因为至少要达到一个月，一个单词才达到“成熟”。再例如，大过年的，暴饮暴食，缺乏锻炼：胖两三斤，就要花一个多月来减肥。

我先前对习惯的要求很低：认为特殊的日子，特殊对待；偶偶放弃，无关紧要。于是，很多好的习惯慢慢废掉。正是这些偶偶的放弃，才有了完全的放弃。虽然已经养成一个月以上的习惯，但是仍然不能丝毫放松警惕。

title: 自律与节制

节制可以看作是自律的一部分，而且我认为节制非常重要。比如说，你对某样事情特别感兴趣，或者你的研究有突破性进展，或者你非常喜欢吃某样食物，或者你很想减肥等等，即使这些目的很好，但是都得节制。年轻人做事情一般都比较疯狂（无节制
）， 对于自己喜欢的事情或者事物会发很多精力去追求。至少我是这样子。

提高意志力，养成好的生活习惯，好的时间管理可以使我们更加自律。这方面的书也很多，但做起来难度很大。希望自己好好加油。

into consideration the causes of said motion. It doesn't ask "how did
the velocity of the body change?" Only, "by how much did it change?"

velocity change.

title: 地震学大挑战
author: imlarry
date: 2015-7-16
tags: Seismology
slug: SEISMOLOGICAL-GRAND-CHALLENGES

categories: Geophysics

翻译自SEISMOLOGICAL GRAND CHALLENGES，选取其中1，4，8，IRIS2008年撰写。

大挑战 1 断层是怎么滑移的？

普通大众主要将地震学与地震联系在一起，把它作为最广泛认可的地球科学学科之一。理解地震断层的本质一直将是最优先的地震学任务，它对社会造成非常多的影响。地球构造板块的相对稳定运动集中了应力，应力主要通过在板块边界或者板块内部滑动来释放。多尺度断层区域系统，从微观尺度到板块边界尺度，都涉及到地震触发，破裂，终止。通过断层在这些复杂系统中的滑移，来寻找对非线性过程的详细物理理解是一个需要解决的地震学大挑战。

断层的滑动运动表现为一系列的复杂行为。最壮观的景象发生在常规地震的应力释放中。由于相邻板块的相对运动造成的，在几百年到几千年间存储在岩石中的弹性势能，在克服断层摩擦阻力时，会被迅速释放。局部剪切运动发生在几秒时间内，产生向外辐射的地震波。近代观察解释了一系列额外的断层滑动行为，从没有明显阻力的稳定滑动，到偶尔滑动的断层，从一系列许多叠加事件的震颤，到超剪切速度滑动、
发射地震激波(发射能引起具有很大破坏力的地面运动)的震颤。

地震学为调查断裂带提供了许多具有最高分辨率的工具。
即使现代模型做出了许多简化假设，例如断层是平面的，滑动式单向的，这些假设可能不总是正确的；地震记录仍可以提供在不同环境中的断裂带的几何图像与依靠时间的属性的图像。沿断层带的流体浓度和压力变化在摩擦行为中起到重要作用，而致力于地震手段获得流体分布图像的努力已经获得成功。从极小到中等地震的系列位置目录，精确到几十米，解释了在断层和沿着单一断层面的不同摩擦行为。在断层上的小地震的连续排列已经通过精确的地震事件位置所发现，在断层的相同位置重复发生的一致的地震的许多例子已经被研究。全球与区域地震台站组合、深井眼地震仪器(例如地球透镜计划SAFO项目的井眼)，提供了捕获断层破裂的触发、生长、终止的记录运动学和动力学断层模型，以物理为基础的理论模型为约束，用来预测强震动，至少在一定概率意义上。过去十年，在最令人兴奋的地球科学发现中，有慢地震和地震震颤的耦合现象。慢滑移过程似乎代表了在稳定滑动和粘滑地震之间的摩擦行为。地震断层震颤，一种低程度、长周期的地震震颤，在一些环境中与慢滑移有关，可能是许多单一亚事件的叠加，但是它的本质仍旧在调查。

地震学已经对断层可能如何破裂和在哪里破裂的基础理解取得了重大进步。但是现在没有可靠的方法来做即将发生的地震的短期预警。获得的观察已经为陆地使用规划提供了游泳的地震灾害评估，作为建造标准的指导，和紧急响应的规划。通过强化观察和理解地震破裂物理过程，更多的事情可以被实现，从更好的预测地面震动，到早期地震和海啸灾害的预警系统的扩展。

关键问题和议题

• 控制不同类型的断层滑移的物理属性是什么？ •
在断层上的一个点的状态和整个断层面的状态之间的关系怎么演化？ •
断层破裂有准备阶段吗？破裂怎么停止？ •
板块间和板块内部的地震机理是不同的吗？ •
控制断层多样性的摩擦本构关系是什么？对于高速滑移的摩擦属性是怎么不同的？什么因素控制了从粘滑到稳定滑动的转变？
• 高应力粗糙面和摩擦力变化的基础本质是什么？ •
幕式断层震颤和滑移是怎么与大地震发生联系在一起的？ •
地震破裂区恢复和重载是怎么发生的？ •
大地震和小地震有什么一般区别(如果有的话)？ •
破裂方向和相应的地面运动能够通过材料性质被预期吗？ •

• 地震震级能够多快的被确定？可靠的地震和海啸预警发布可以多快？

• 在断层附近的浅井眼中部署更灵敏和低噪声的地震台站； •
在震后用大量可便携仪器快速部署； • 收集更多大地震的强的地面运动记录； •

增加与其他学科的交流与合作，例如钻井工作，地质研究，实验室研究。

大挑战4 海气过程怎样与固体地球作用？

地震学容易探测自然震源的信号，例如海洋风暴、
火流星、龙卷风、冰川崩解。一个新的研究时代最近已经打开在固体地球物理、冰川学、物理海洋学、大气科学面前，具有变化的科学和社会关联的高度可能性。海气过程怎样融入到固体地球中的地震波，以及利用这些怎样来检测全球环境的多学科交叉课题，是一个高度优先的地震学大挑战。

理解海洋波动和地震波在宽频率范围内的耦合机理很有趣，因为它能检测海洋过程，反过来，海气能量与固体地球的耦合为探索地球内部结构提供了一个新来源。地球长周期的“嗡嗡声”，或者几百秒周期的地球自由震荡的连续激发，
在十年前，已通过全球地震台网中过去几十年积累的高质量、连续的记录而发现。现在已经达成共识，嗡嗡声的初始来源与产生强海洋波动的中纬度冬季暴风有关，强海洋波动通过不清楚的非线性机理与海底耦合。

海洋波动起源于短周期(4-30s)微震背景在19世纪50年代已经被说明，但是几十年来，这个无处不在的信号被广泛认为扰人的噪音。事实上，微震是一种独特的遍及全球海洋的风暴能量的整合。地震学家和合作者现在正致力于弄清楚地球地震背景激发和海洋波动、大气过程之间的关系。这个研究是大量、多学科交叉群体感兴趣的对象，应用范围可以从地球构造到对浮冰、沿岸物理海洋学的影响(例如在海港的长周期海洋波动的影响)。地震学正在提供一个新的、有价值的综合窗口，来探究其他手段无能为力的全球气候变化尺度。全球变暖影响到大尺度大气循环模式，造成暴风周期和强度的变化。微震和嗡嗡声能够跟踪海洋风暴和他们与海岸的波浪作用。监测波浪活动的变化，识别是否这些变化发生在过去世纪的波浪系统中，特别是南大洋，可能可以从海岸线附近地震台站的存档的地震图中得到。最近发现地震方法能够探测到在水柱的分层和混合。具有前所未有的高水平精度的海洋结构能够被用来内波能量和湍流混合的定量估计，能够帮助说明温盐循环，温盐循环在气候和大气二氧化碳的自然封存方面起到重要作用。

地震学家和冰川学家正在合作致力于追踪基地冰盖是怎么被全球变暖影响的。冰川地震，由量达数立方公里体积的冰块在数十秒内突然移动造成，一直未引起关注，直到最近，因为冰川地震不产生短周期(1-10s)远距离可见的能够用来标准地震探测和定位的地震波。冰川震源涉及到浮冰系统，例如裂冰，激发像飓风一样的海浪，能够被部署在陆地和浮冰上的地震仪所探测到，为监测与入海冰川和冰架稳定性相关联的关键过程提供了新机会。

固体地球内部震源产生的波动，不仅在地表传播，而且会通过海洋(海啸、T-阶段)、大气(由火山爆发和地震产生的次声波)，甚至在电离层传播，在那里通过GPS和雷达技术的遥感为大地震震源的特征化提供了新途径。在地球表面的爆炸或扰动产生了地震和次生能量，后者在微气压计，高频，麦克风上可以被观测到。大气现象包括飓风、行星影响、雷击，可以被地震和次声波传感次检测到，为这些过程和它们的全球出现提供了新的限制。也许把地震和次声波监测联合来探测和量化磷火，使用类似的策略到火山爆发。地震记录也可以感受到引起地面倾斜的大气压力变化，例如在洛杉矶和澳大利亚少见的牵牛花云的形成。将地震和次声记录相结合能够帮助理清声波在大气中传播的方式，从而更好地理解大气结构和其他方式不能够解决的空间和时间尺度变化。

大挑战8 板块边界系统怎么演化？

发生在板块边界的过程造成了一些文明的巨大自然灾害——大地震和活动火山。板块边界过程也强烈影响了广泛的地球表面，驱动了类似于造山、大陆岩石圈的岩浆演化、弧后裂谷、由陆块碰撞引起的地壳挤压等全球事件。一个地震学起着主要作用的大挑战是理解发生在板块边界的广泛分布的变形区多尺度过程，以及它们是怎么和自然灾害和这些区域的地质联系在一起的。

协调的地球物理网络包含不同类型的传感器，可以用来理解大陆和海洋区域的广泛的板块边界系统的动力学过程。地震仪器为决定地震位置、断层平面方向、探测低频震颤和板块结构成像提供了数据。大地测量仪器能够用于在不同的应变率下，应力积累和释放的成像，
并提供了约束板块流变学的变形的记录。现代地震和大地测量观察的快速增长允许我们识别地质变化在驱动板块断层系统、变形和震级的发展中扮演的作用。

地球透镜计划项目是当代使用综合地球物理观察和分析来研究板块边界系统的一个实例。地震仪、GPS台站、应力计的密集分布，这作为美国台阵和板块边界观察的一部分，正在提供数据，转变了我们对板块边界的认识。例如，使用地球透镜计划的观察，在西北太平洋的集中研究，正开始揭示胡安德富卡板块俯冲的历史，以及它与遍及美国西部的活跃地质构造过程的联系，包括黄石热区，大盆地，格兰德河裂痕。圣安德烈斯断层系统是地球上研究最多的板块边界，有广泛的地震、应力、钻井、地形和大地测量数据收集工作。通过地球透镜计划，地震学家正在对板块边界的深部构造进行非常详尽的成像，解释了地壳变形模式是如何与岩石圈与软流圈的构造、组成、物理性质相联系。

大洋板块边界与大陆板块边界一样复杂、弥散，但是目前我们只有相对很少的数据来约束它们的深部构造和过程。在洋中脊的融化是一个被动过程，相应于远距离板块的力。新产生的地壳有统一的7公里厚度。上涌地幔的部分熔融开始于100公里深度左右。因此大洋地壳形成过程驱使熔融区的大量地幔的循环，产生在地壳和下覆残留地幔(冷却成大洋岩石圈)之间的组分各向异性。
通过这个过程产生的融化途径仍然神秘，主要由于目前用于研究海底过程的有限的数据和仪器。需要更多的仪器部署来更好的约束和改进设计海底扩张过程的小尺度和大尺度结构的理解。

在有密集地球物理仪器部署的板块边界区域，地震学家现在能够定量化初始的结构单元和地质单元的变形和成像。通过施加在板块边界的力与它们的结构和流变学的耦合，我们能够开始建造这些系统的动力学模型，更好的理解地球板块构造系统的基本过程。

title: 克制、计划与有规律的生活
author: imlarry
date: 2015-06-26

tags: life, Procrastinate

最近对于有规律的生活是越来越向往，一切都以这种目标中的生活为中心，当然偶偶也有例外。在追求这种规律的生活中，我遇到了一些困难，这些困难反复出现，具有一定的特征；虽然我未统计这些困难的各项指标，但是在感性上已经有了一些认识，所以我必须要来写一写。

那么这些困难是什么呢？是无法克制自己。是出现了计划外的事情。为什么我要写，因为我觉得这两种困难，在很大程度上是可以避免的。当然，不可能完全避免，不可能完全克制，自然更不可能不出现计划外的事情。

首先，说一下克制。我对一个知乎的问题非常深刻：男人品位的高低从何处体现。排行第一的答案是“我所观察到有品位男人的最高体现，私认为是五个字——分寸与克制”。当时我是不太赞同的。现在呢，我极其赞同。对的。克制太难了。我总是克制不住玩手机，总是克制不住刷blog。我每天要在这些事情上浪费多少时间，虽然没有统计，但我估计有1个小时，其中大半个小时是可以避免的。比如说午睡、早上醒来，总是会先玩一下手机，这种习惯就非常不好，不仅浪费了时间，还有可能导致再次睡觉。值得高兴的是，我戒掉了晚上睡觉前玩手机，但是这种陋习竟然慢慢转移到了早上、午睡。还有的是在学习的过程中，总是分心。当然这和学习环境有关，为了有一个好的学习环境，教室是最佳的。

其次，我说一下，计划。计划很重要，总体来说，计划都是要提前的。但是我有时候没计划，或者有计划的话，也不怎么提前。这样导致的一个后果是，如果有突发事件，我就忙不过来，然后就很有可能会熬夜，大大影响生活节奏。就先前的计划来说，我的计划一般在前一天晚上或者当天早上制定的。这样的计划，虽然比较容易完成，但是不可避免突发事件。长远的计划往往执行性非常差，因为你不可能预计3~4天之后的事情。我的想法是将GTD与日记本结合，GTD里记录将来要做的事情，笔记本里记录每一天具体的计划。

Summary Tips

  1. 早上、午睡起来不玩手机
  2. 去自习室自习
  3. GTD与日记本结合

Seismic method is a high-tech method. It uses receivers to record
generated seismic waves. It is used wildly in a number of fields, such
as prospecting for oil and probing the internal structure of the Earth
and ocean.

Prospecting for oil

The most important procedure of prospecting for oil is a seismic survey.
It tells the geologists the underground information and the possibility
of oil existence. The method requires a controlled seismic source of
energy, such as dynamite or a seismic vibrator. The size and scale of
seismic surveys has increased with the development of computer
technology.

Probing internal structure of the Earth

A seismic method is the only way to probe the internal structure of the
Earth. The elastic waves excited by earthquakes, refracted and reflected
within the Earth, could be used to probe its depths and gather
information on the elastic structure and eventually the physics and
chemistry of inaccessible regions down to the center of the Earth.

A seismic method also helps to probe the internal structure of the
ocean. The theory of probing the internal structure of the ocean is just
like probing the internal structure of the Earth. The deep ocean is so
deep that nobody can touch it. Besides, humans would spend lots of time
to probe a small part of ocean because oceans are too enormous. Hence,
the presence of seismic method helps the quick investigation of ocean
with high accuracy.

Seismic method is an extraordinary and very high-tech method. It can be
applied to many important areas. I think it helps a lot to create modem
industry base because of the oil prospecting. And I hope that the method

author: imlarry
date: 2014-12-29
tags: Geophysics

在介绍主题之前，先来谈一个非常重要的数学思维方法：几何方法。在大学之前，我们学习过一次函数、二次函数、三角函数、指数函数、对数函数等，方程则是求函数的零点；到了大学，我们学微积分、复变函数、实变函数、泛函等。我们一直都在学习和研究各种函数及其性质，函数是数学一条重要线索，另一条重要线索——几何，在函数的研究中发挥着不可替代的作用，几何是函数形象表达，函数是几何抽象描述，几何研究“形”，函数研究“数”，它们交织在一起推动数学向更深更抽象的方向发展。

函数图象联系了函数和几何，表达两个数之间的变化关系，映射推广了函数的概念，使得自变量不再仅仅局限于一个数，也不再局限于一维，任何事物都可以拿来作映射，维数可以是任意维，传统的函数图象已无法直观地表达高维对象之间的映射关系，这就要求我们在观念中，把三维的几何空间推广到抽象的n维空间。

由于映射的对象可以是任何事物，为了便于研究映射的性质以及数学表达，我们首先需要对映射的对象进行“量化”，取定一组“基”，确定事物在这组基下的坐标，事物同构于我们所熟悉的抽象几何空间中的点，事物的映射可以理解为从一个空间中的点到另一个空间的点的映射，而映射本身也是事物，自然也可以抽象为映射空间中的一个点，这就是泛函中需要研究的对象——函数。

从一个线性空间到另一个线性空间的线性映射，可以用一个矩阵来表达，矩阵被看线性作映射，线性映射的性质可以通过研究矩阵的性质来获得，比如矩阵的秩反映了线性映射值域空间的维数，可逆矩阵反映了线性映射的可逆，而矩阵的范数又反映了线性映射的哪些方面的性质呢？矩阵范数反映了线性映射把一个向量映射为另一个向量，向量的“长度”缩放的比例。

范数是把一个事物映射到非负实数，且满足非负性、齐次性、三角不等式，符合以上定义的都可以称之为范数，所以，范数的具体形式有很多种（由内积定义可以导出范数，范数还也可以有其他定义，或其他方式导出），要理解矩阵的算子范数，首先要理解向量范数的内涵。矩阵的算子范数，是由向量范数导出的，由形式可以知：

title: 地球物理专业的计算方法
author: imlarry
date: 2014-12-29
tags: Geophysics
slug: computation_method_of_geophysics

categories: Geophysics

文章转自康建，部分修改

现在地球物理的研究和应用主要靠“算”。地球物理专业本科一般开计算方法课，课程的理论基础主要涉及数值分析、数据处理、或数字信号处理等内容，实践部分一般是用fortran、C语言或Matlab实现相应的算法。

涉及的基础内容一般包括：

  1. 误差分析。
  2. 插值（拉格朗日插值、均差与牛顿插值、埃尔米特插值、分段低次插值、三次样条插值）。
  3. 函数逼近与曲线拟合（函数逼近的基本概念，函数空间、范数、内积、最佳逼近，正交多项式，曲线拟合的最小二乘法，三角多项式逼近、快速傅立叶变换）。
  4. 数值积分与数值微分(牛顿-柯特斯公式、复合求积公式、龙贝格求和公式，高斯求积公式等，多数还是要用到插值和拟合的理论)。
  5. 解线性方程组，有直接法、迭代法。直接法多数涉及到矩阵分解计算的理论和方法，如高斯消去法、三角分解法等，通过研究矩阵范数、条件数等理论来改进算法；迭代法有雅可比迭代法、高斯-塞德尔迭代法、超松弛迭代法、共轭梯度法等，都用到矩阵和向量的运算。
  6. 非线性方程和方程组的求解，多数用迭代来解决，有二分法、牛顿法、弦截法、抛物线法等，看似简单基本的方法，但是如何加速收敛涉及的问题就复杂了，研究非线性不动点、收敛性就有可能联系到混沌了，有些分形图就是不断递归画出来的。
  7. 矩阵特征值计算，矩阵的理论和计算是个大问题，很多数值问题最后都归结为矩阵计算，除了上述解线性方程组，求特征值在地球物理中也常用，连续介质力学问题离不开特征值，其实物理学中特征值问题几乎哪儿都有，研究主应力、波动方程用到，听说量子力学中也用，没学过啊。基本常用的算法有：幂法、反幂法、QR法，矩阵分解和正交变换要搞清楚。就矩阵来说，还有更进一步的奇异值分解，奇异值分解对地球物理反演有重要意义。

下面话一下“微分方程”，高等数学中学了点常微分方程求解，又专门学了数学物理方程求解，当费很大力气学了各种方法解各种类型的偏微分方程后，到了计算方法课才真正体会到学了那么多方法也只是能解一小部分问题，只能靠数值方法求近似解，所以微分方程数值解是计算方法的重要内容。

常微分方程初值问题数值解法简单的有欧拉法、梯形法等，龙格-库塔法是常用的有效方法，这些也是插值理论的应用，还有多种线性多步法等，还有方程组的求解，有的也用到打靶法和有限差分。

偏微分方程的解法常用的有有限差分法、有限元法等，也是现在研究的热点。

以上是基本的内容，在传统的《数值分析》，或较简偏工科的《计算方法》或《科学与工程计算》中都有，有的理论也不多讲，直接用Matlab、C、Fortran或Python来实现算法，当然一本书也囊括不了上述内容，有的内容只是提到，如有限元，一章内容是讲不清学不明的。当然以上内容也不全，有时也会用到最优化数值方法，线性规划等，有的教材就包含这些。

数字信号处理是单独的一门课，但有时也归为地球物理计算方法，数字信号处理和数值分析这两门课有很多地方是紧密联系的，如快速傅立叶变换，都是其中的重要内容。地球物理的计算确实很多都是对波的处理，数字信号的处理的理论和算法是必不可少的。

除以上一般理论和方法外（这些在其它科学中也用），地球物理还有很多相对专用的数值计算方法。如地球物理反演就是很重要很丰富的内容，是研究地球内部结构和勘探的主要手段。

朱介寿先生《地震学中的计算方法》、Durk J.
Doornbos的《地震学算法理论-计算方法和计算机程序》书中提到以下内容：解反演问题的基本数值方法（泛函、广义逆、奇异值分解、计算目标函数导数的方法、广义反演线性方程组的修正、分辨矩阵、信息矩阵和协方差矩阵），约束条件下反演问题的数值解法（约束条件下最小二乘问题解法、惩罚函数法、可变容限法），震源断层面解的数值方法（P波初动解方法，P波初动断层面解计算），震源的定位（Geiger法、Powell法修定震源参数、非均匀地壳模型中地震波走时计算），层状介质体波走时计算（反射波走）、用地震体波反演地球内部速度分布，用地震面波研究地球内部结构，理论地震图的计算，任意介质中的射线追踪方法。

Durk J.
Doornbos的《地震学算法理论-计算方法和计算机程序》书中介绍了以下内容：介质中的射线和体波（垂直向变化层构成的地球模型中的体波谱合成、WKBJ地震图算法、非球状和椭率修正），三维介质中的射线和体波（三维结构中完全射线追踪，射线追踪和波前的数值方法），层状介质中的反射率法（地震波场的系统近似方法，反射率计算中的混合法，波数积分求积），成层介质中的振型法（计算地震简正振型解，地球和太阳自由振荡的本征频率和本征函数的计算，变分法），通用算法和震源（矢量和矩阵运算算法-基矢量运算、非稀疏矩阵的运算、稀疏矩阵反演算法，矩张量源的辐射-体波面波辐射花样，初动资料的单一和联合断层面解-概率模型和似然函数。

title: Nomenclature for Sequence Stratigraphy
author: imlarry
date: 2014-12-31
tags: Geophysics
slug: nomenclature_for_sequence_stratigraphy

categories: Geophysics

  1. Allogenic processes : externally controlled process, such as sea level, climate, and/or platform-wide subsidence patterns
  2. Autogenic processes : inherent to the sedimentary system, such as lateral migration of sediment bodies.
  3. Base level : hightest level to which a sedimentary succession can be built, or lowest level of continental denudation.
  4. Accommodation : the amount of space available for sediments to fill, measured from the seafloor to the base level(positive/negative).
  5. Aggradation/Erosion : postive/negative fluival accommodation, respectively.
  6. Transgression/Regression : landward/seaward shoreline.
  7. Retrogradation/Progradation : landward/seaward shoreline.
  8. Denudation : the long-term sum of processes that cause the wearing away of the earth’s surface leading to a reduction in elevation and relief of landforms and landscapes. eg.weathering, erosion, and mass wasting.
  9. Relative sea level : distance between the datum and the sea level.
  10. Lowstand system tract(LST)/Highstand system tract(HST) : the early (lowstand) and late (highstand) stages of base level rise.
  11. Transgressive system tract(TST) : transgressive deposits
  12. Regressive system tract(RST) : LST+HST+forced regression
  13. Clinoform : used to describe the more steeply dipping portion of the basin-margin profile (commonly >1°) developed basinward of the topset. Clinoforms generally contain deeper water depositional characteristic of slope. The slope of the clinoform usually can be resolved on seismic data.
  14. Shelf Break : The shelf break marks a distinct change in the slope of the sea floor from the flat continental shelf to the steep sea floor of the continental slope.

title: 跨度学习

DFT原理

设$x(n)$为一长度为N的序列，$x(n)$的DFT变换为:

$$X(k) = sum_{0}^{N-1} x(n) W_{N}^{nk} R_{N}{(k)}$$

按时间抽选的基-2
FFT算法是DFT的一种快速算法。其难点倒位序的实现和$W_{N}{r}$的计算。

Python程序

#!/usr/bin/python
def FFT(xn):
    N = xn.__len__()
    xn = xn*(1j)
    xn = xn*(1j)
    xn = xn*(-1)
    NV2 = N/2
    J=0
    for I in xrange(N-1):
        if(I<J):
        T=xn[I]
        xn[I]=xn[J]
        xn[J]=T

        K = NV2
        while( K <= J ):
            J = J - K
            K = K/2
        J = J + K

    L = int(np.log2(N))
    for M in xrange(1,L):
        LE = np.power(2, M)
        LE1 = LE / 2

        U = 1
        W = complex(  np.cos(np.pi/LE1), np.sin(np.pi/LE1) )
        J = 0

        for J in xrange(LE1):
            I = J
            while ( I < N ):
                IP = I + LE1
                T = xn[IP]*U
                xn[IP] = xn[I] - T
                xn[I] = xn[I] + T
                I = I + LE
            U = U * W

    return xn

开始冥想

  • 选择合适的坐姿。双脚平放在地上，或盘腿坐在垫子上。背挺直，双手放在膝盖上。注意舒适即可，不需要做成标准的莲花式。

title: 基础训练计划
date: 2014-08-15
author: imlarry

tags: 健身

循序渐进

如果你的肌肉一直承受一个它们还不习惯的阻力，那么你就能要使它们逐渐适应那个阻力，并变得更强壮。但是，一旦它们完全适应了，这个生长过程就会停止。当这种情况发生时，能使你的肌肉继续生长的唯一办法就是进一步增长它们所承受的超负荷的量。并且阻力的增加必须循序渐进。

反复

一次反复是指一个练习动作的一个完整循环——一次肌肉的收缩，继而一次肌肉的伸展。一个练习组是指许多次这样的反复。

研究和经验显示，在每个练习中使用的重量是它能承受的最大重量的70% ~
75%时，训练效果是最好的。

练到力竭

在健身中，
练到力竭并不是指训练到完全耗尽力气。它仅指你不断进行一组练习，直到最后如果不休息的话，再也不能做一次反复。这是由于ATP和氧气的消耗，肌纤维不能再次收缩，这时肌肉就需要休息和复原。

练到力竭为什么这么重要？因为当你使用比单次极限重量稍轻的重量进行动作反复时，所有的肌纤维不是立马开始工作。这时你只有使用到一部分肌纤维，那么等它们变得疲惫之后，身体会用另外一些肌纤维来替换疲惫的肌纤维。将一组动作练到力竭，是一种使用到所有的肌纤维的方法。

有了一定经验的健身者，针对上半身就会这样训练:

  • 第一组： 使用较轻重量的热身组，15次反复或者稍微多一点；
  • 第二组：增加重量，使得肌肉只能做大致10~12次的反复；
  • 第三组：继续增加重量，使得力竭出现在8~10次反复后；
  • 第四组：增加足够的重量，使得肌肉只能做6次反复（强力组）
  • 第五组（可选）：使用相同重量，尽量再做6次反复。如果需要的话，向训练伙伴寻求帮助（被迫强迫组）。

练习组

一般而言，在基础训练计划中，我推荐每种练习动作都包括4个组，有特别规定的除外：

  • 你需要至少做4组动作。
  • 每种动作进行4组。大肌肉群总共进行12组，在高级的训练程序中则是20组。
  • 健身者们50年的训练经验证明，你能承受的、可以让你完成4组训练的最大重量可以刺激肌肉，让他们生长。

动作幅度

在大多数情况下，健身练习应该以最大的动作幅度来训练肌肉。你应该注意伸展到最大限度，然后原路返回，达到完全收缩的状态。

肌肉收缩的质量

为了有效的进行训练，你必须孤立训练不同的肌肉。另外，还要注意选择正确的重量。一旦你使用的重量对你来说过重的话，你的身体会自动召集其他肌肉工作，这就是神经系统的工作机制。所以，你能举起某个重量，并不代表你做的动作是正确的。

如何才能做到这点？一种方法是，开始的时候使用非常轻的重量，将注意力集中在整个运动过程中肌肉的感受上，渐渐地增加重量。当你使用的重量增加到某个点，让你感觉和使用较轻重量时肌肉的工作方式有所差异时，那么很可能你使用的重量过重，你应该减轻些重量，重新找回以前的那种感觉。

热身

热身可以将充氧的鲜血输送到身体的相应区域中，可以最大化给身体供给氧气，并有助于消除肌肉工作时产生的废品。

热身的方式多种多样，可以进行一会儿心肺训练（跑步，单车，健身操等）。也可以使用重量本身。

强力训练

在你的训练程序中加入几个强力组，这也可以帮助你变得更加强壮。经过强力训练的肌肉可以坚如磐石，因为这增大了肌纤维的尺寸。

重磅日

每周或者每隔一周，选择身体的某个部位，检验一下你的极限力量。训练开始时，拉伸一下身体，并进行热身运动。在你的训练日志中记录你举起的重量。这也极大地有助你的自信心增强，并更加投入地进行训练。

训练过度与恢复

如果你过度频繁地训练某块肌肉，使其无法完全恢复，就会导致训练过度。不同肌肉恢复的速度是不同的。肱二头肌是恢复最快的肌肉，下背部的肌肉恢复得最慢，需要花100小时才能从大重量训练中完全恢复过来。在大多数情况下，让肌肉休息48小时就足够了。训练过的肌肉比没有训练过的肌肉恢复的更加快。所以，你的水平越高，你恢复的速度就越快，同时你的训练强度就可以越大。

呼吸

简单的法则：用力时呼吸，不要屏住呼吸。肌肉在强力收缩时常常伴随着隔膜的收缩，特别是进行推举或者深蹲时，会增加胸腔的压力。如果你屏住呼吸，可能会导致受伤。

拉伸

拉伸可以增加肌肉、肌腱和韧带的柔韧性，提高训练时的安全度，防止过度拉伸。除了训练前、训练后做拉伸以外，在训练中也应该加入一些拉伸动作。拉伸时动作要缓慢、轻柔，保持30秒或者更长时间。如果突然给肌肉或者肌腱增加很大压力，它们会自动收缩，以自我保护。

肋骨外翻

肋骨外翻，学名上叫做“肋间内肌功能丧失”。当你平卧的时候，最下缘的肋骨超过身体的外缘，就叫做肋骨外翻。肋骨外翻一般是幼年发育时缺钙造成的。但是如果你的健身动作不正确，也会导致肋骨外翻。常见的是在健身中不会用腹横肌来收腹，从而形成肋骨外翻。

健身误区

我们训练中常见的肋骨外翻误区：让肚脐贴向脊椎。上瑜伽或普拉提课程时，我们总听到一句话，说：让你的肚脐贴向脊椎。但是大多数人腹部力量不足，当他听到这个口令时，会先吸肚子肋骨上提，用腰部肌肉向前一顶完成动作，而不是腹肌发力向下沉肋。

造成的影响

  1. 无法收紧核心，导致上肢和臀部练不好:如果你不是收紧腹横肌而是用腰部发力，肋骨外翻去做动作的时候，确实力量会变大，但那不是目标肌肉发的力，如果是上肢的话，胸部是最不容易练出来的；下肢则都去练大腿而练不到臀部了。
  2. 腰椎压力大:因为肋骨外翻，导致整个脊椎过直，从而失去了脊柱的正常生理弯曲，也就失去了脊柱减震的作用 ，很容易让脊椎受到伤害。
  3. 脊椎灵活性变差 :长期的肋骨外翻，会让你的胸椎变直，脊椎灵活性的失去，会让身体循环功能变差。
  4. 肾功能的影响 :肋骨外翻，压迫到胸椎11、12；而这个部位的植物神经刚好支配肾脏。

正确收腹

在健身中经常听到“收腹”两个字，但是你真的会收腹吗？大多数人通过吸肚子来收腹，你现在就可以平躺着尝试一下，你会发现胸腔隆起了，腹部还是松的。正确的收腹方法是用腹横肌来收缩。腹横肌是腹部肌肉中最深层的肌肉，包裹在下腹部周围，链接到腰背部。当腹横肌收缩时，能稳定脊椎病支撑腰背部。

正确收腹步骤：

  • 平躺并保持脊椎自然中立位
  • 收缩腹横肌，保持正常呼吸
  • 肋骨保持放松，整个过程不能提升
  • 这时找到骨盆前侧突出的骨头，手指放在骨头位置的内侧向下按住，会感觉到肌肉的收缩。

改善

据不完全统计，注意肋骨外翻的教练不足5%，而即使是普拉提运动最发达的美国，普拉提教练中也仅仅只有10%会注重这个问题；会员中基本上大多都存在肋骨外翻问题（引自香港体协HKSCA）。

  • 肋间内肌训练：普拉提肋式呼吸法
  • 腹横肌训练：核心力量的加强
  • 提高肩关节柔韧性：肩关节越紧的人，肋骨越容易外翻，在训练前我们要适当地放松和伸展肩关节。

A list is a collection of items. Each item holds a relateive position
with respect to the others. We also refer to this type of list as an
unordered list.

In order to implement an unordered list, we use linked list to
construct lists.

The node class

We use node to construct linked list. Each node object contains two
kinds of information: the data and the reference to next data.

The interface of class node contains several operations as follows:

  • __init__ : initialize a new node
  • getData : read data from the node
  • getNext : get reference of next node from the node
  • setData : set or reset data of the node
  • setNext : set or reset reference of next node
class Node:
    def __init__(self, initdata):
        self.data = initdata
        self.next = None

Using a stack to evaluate postfix

Here, we use a stack ADT to evaluate postfix. Postfix make operator
follows operands, such as 6 7 + . Using a stack in that way:

  1. split the expressions to numbers and operators
  2. read a number or operator each once
  3. push a number on a stack after reading a number
  4. pop two numbers off stack and calculate them after reading a operator

In order to search a number, we use regular expression(re). Python

import re   
import string

class Stack:

        self.items = []

    def pop(self):
        self.items.pop()

    def push(self, item):
        self.items.append(item)


def postfix(expr):
    if expr == "":
        print "The expression format wrong!"

    expr_split = expr.split(' ')
    expr_stack = Stack()

    for char in expr_split:
        if char == '+':
            expr_stack.push(expr_stack.pop() + expr_stack.pop())

1.1Cython基础

Cython的基本本质可以总结为：Cython是带有C数据类型的Python。
Cython是Python：几乎所有Python代码在Cython中都有效。Cython编译器把Python代码转换为C代码。
但是Cython不仅仅局限于此，因为Cython的变量和参数可以声明为C数据类型。处理Python和C值的代码可以自由混合，并且代码之间的转换是自动的。引用计数维护和Python操作符的错误检测都是自动的，并且处理（包括try-except
和try-finally
语句）组件的Python异常都是可用的，甚至在处理C数据中间时。

4扩展类型

为了支持面向对象编程，Cython支持常规的Python类，与在Python中完全一样：

class MathFunction(object):
    def __init__(self, name, operator):
        self.name = name
        self.operator = operator

    def __call__(self, *operands):
        return self.operator(*operands)

虽然Cython基于Python的内建类型(built-in
type)，但是，Cython也支持第二种类：扩展类型，有时候被称为cdef类型，由于用于声明的关键词。相比于Python类型，它们有些限制，但是比一般的Python
类型有更高的内存使用效率和速度。它们之间的主要差别是扩展类型使用C
struct存储字段和方法，而不是Python字典。这允许扩展类型在字段中存储任意C类型，而不需要Python包装器，并且直接从C级别传递字段和方法，而不需要通过Python字典查找。

常规的Python类能够从cdef类继承，但是不是倒过来。Cython需要知道完整的继承层次来安排C
struct，并且限制为单继承。常规的Python类，另一方面，可以从任何Python类型和扩展类型，可以在Cython代码或者纯Python代码中。

目前为止，我们的积分例子还没有完全游泳，因为它只集成了单一的硬编码函数。为了改进，几乎不牺牲速度，我们使用cdef类型来代替函数：

cdef class Function:
    cpdef double evaluate(self, double x) except *:
        return 0

cdef class Sin0fSquareFunction(Function):
    cpdef double evaluate(self, double x) except *:
        return sin(x**2)

def integrate( Function f, double a, double b, int N):

        raise ValueError("f cannot be None")
    s = 0
    dx = (b -a)/N
    for i in xrange(N):
        s += f.evaluate(a + i*dx)
    return s*dx

def Find_Max_Crossing_Subarray(A, low, mid, high):
    left_sum = -float('inf')
    max_left = 0
    subsum=0
    for i in range(mid, low, -1):
        subsum = subsum + A[i]
        if subsum > left_sum:
            left_sum = subsum
            max_left = i

    right_sum = -float('inf')
    max_right = 0
    subsum=0
    for j in range(mid+1, high):
        subsum = subsum + A[j]
        if subsum > right_sum:
            right_sum = subsum
            max_right = j
    return (max_left, max_right, left_sum+right_sum)

def Find_Max_Subarray(A, low, high):
    if high == low:
        return (low, high , A[low])

    else:
        mid = int((low+high)/2)
        (left_low, left_high, left_sum) = \
            Find_Max_Subarray(A, low, mid)
        (right_low, right_high, right_sum) =\
            Find_Max_Subarray(A, mid+1, high)

            return (left_low, left_high, left_sum)
        elif right_sum >= left_sum and right_sum >= cross_sum:
            return (right_low, right_high, right_sum)
        else:
            return (cross_low, cross_high, cross_sum)

下面为最大子数组问题的非递归的、线性时间的算法
。从数组的左边界开始，由左至右处理，记录到目前为止已经处理过的最大子数组。若已知A[1…j]
的最大子数组，基于如下性质将解扩展为最大子数组：A[1..j+1]
的最大子数组要么是A[1…j] 的最大子数组，要么是某个子数组A[1..j+1]
。在已知A[1…j]
的最大子数组的情况下，可以在线性时间内找出形如A[i..j+1] 的最大子数组。

#!/usr/bin/python
#Filename: find_max_subarray_linear.py

def Find_Maximum_Subarray_Linear(A):
    max_so_far = max_ending_here = A[0]
    tlow,high,low = 0,0,0
    for i in xrange(1,len(A)):
        if max_ending_here < 0:
            max_ending_here = A[i]
            tlow = i
        else:
            max_ending_here += A[i]
        if max_ending_here > max_so_far:
            max_so_far = max_ending_here
            low = tlow
            high = i

def Counting_Inversions(seq):
    if len(seq) <= 1:
        return (seq, 0)
    mid = int(len(seq)/2)
    (left,x)  = Counting_Inversions(seq[:mid])
    (right,y) = Counting_Inversions(seq[mid:])
    count=x+y
    return Split_Count(left,right,count)


def Split_Count(left,right,count):
    result=[]
    leftlen = len(left)
    i,j=0,0
    while i<len(left) and j<len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            count = count + leftlen - i
            j += 1
    result.extend(left[i:])
    result.extend(right[j:])
    return (result,count)
